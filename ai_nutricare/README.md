# 🥗 AI-NutriCare

> **AI/ML-Based Personalized Diet Plan Generator from Medical Reports**

AI-NutriCare reads your medical lab report (PDF or image), extracts all test values using Vision AI, and generates a fully personalized diet plan based on your actual health data, BMI, food culture, and dietary preferences.

---

## ✨ Features

| Step | What Happens | Model Used |
|------|-------------|------------|
| 1 | Upload PDF / JPG / PNG | — |
| 2 | OCR — converts each page to structured Markdown | Llama 4 Scout 17B Vision |
| 3 | Data Extraction — produces structured JSON with all lab values | Kimi K2 Instruct |
| 4 | BMI Calculation | Formula |
| 5 | Personalised Diet Plan Generation | Llama 3.3 70B Versatile |
| 7 | Context-aware AI Chat Assistant | Llama 3.3 70B Versatile |

All models run on **Groq** (free tier available).

## 🚀 Quick Start

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/ai-nutricare.git
cd ai-nutricare
```

### 2. Create a virtual environment
```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Mac / Linux
source .venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Set up your API key
```bash
cp .env.example .env
# Open .env and paste your Groq API key
```
Get a free Groq API key at: https://console.groq.com

### 5. Run the app
```bash
streamlit run app.py
```

Open http://localhost:8501 in your browser.
